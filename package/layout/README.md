This is a basic root directory for ps3netsrv-go. It's layout is identical to [original ps3netsrv layout](https://github.com/aldostools/webMAN-MOD/wiki/~-PS3-NET-Server#content-directories). Copy of table:

<table>
<hr><td>Folder Name<td>Purpose<td>Contains
<tr><td>GAMES<td>Games in folder format (files extracted from BDVD)<td>Folders
<tr><td>PS3ISO<td>ISO images of PS3 blurays<td>ISO + <a href="https://github.com/aldostools/Resources/releases/download/1.0/EP0001-BLES80608_00-COVERS0000000000.pkg">Covers</a><br>ISO + <a href="https://github.com/aldostools/Resources/releases/download/1.0/EP0001-BLES80608_00-COVERS0000000000.pkg">Covers</a> + <a href="https://github.com/aldostools/Resources/blob/main/REDUMP_KEY.7z">.key</a><br>ISO + <a href="https://github.com/aldostools/Resources/releases/download/1.0/EP0001-BLES80608_00-COVERS0000000000.pkg">Covers</a> + <a href="https://github.com/aldostools/Resources/blob/main/REDUMP_DKEY.7z">.dkey</a>
<tr><td>PSXISO<td>ISO images (CUE/BIN) of PS1 CDs.<td>ISO + <a href="https://github.com/aldostools/Resources/releases/download/1.0/EP0001-BLES80608_00-COVERS00000RETRO.pkg">Covers</a>
<tr><td>PS2ISO**<td>ISO images of PS2 DVDs or CDs.<td>ISO + <a href="https://github.com/aldostools/Resources/releases/download/1.0/EP0001-BLES80608_00-COVERS00000RETRO.pkg">Covers</a>
<tr><td>PSPISO**<td>ISO images of PSP UMDs.<td>ISO + Covers
<tr><td>BDISO<td>ISO images of video blurays (decrypted).<td>ISO + Covers
<tr><td>DVDISO<td>ISO images of DVDs.<td>ISO + Covers
<tr><td>ROMS**<td>ROMS folder (zip, 7z, bin, chd, cue, sfc, gb, etc).<td>ROMs + Covers
<tr><td>GAMEI**<td>Folder to store PSN games extracted from packages.<td>Folders
<tr><td>PKG<td>Folder to store PS3 packages. Used to install games, DLC, homebrews, etc.<td>pkg files
<tr><td>MOVIES<td>Folder to store movies (MKV, DIVX, AVI, etc.).<td>Video Files
